## 零、设计摘要

对标矮人要塞：猎物按生态区种群动态刷新（不是一次性生成），尸体留在地上走过去剥皮，腐败时间拉长到有意义的时间尺度，烟熏架终于有用，篝火旁能做熟食。

**核心信念**：食物不该是"永远够"也不该是"天天烂"。有足够时间想"吃还是熏"，过度捕猎会让生态区枯竭。

## 一、猎物生态

### 1.1 三类猎物

| 猎物 | 符号 | 成年 HP | 移速 | AI 性格 | 生成生态区 | max 种群 |
|------|:---:|:---:|:---:|------|------|:---:|
| Deer（鹿） | `D` | 80 | 快 | **逃跑**：见人 10 格内立即逃 | 草原、疏林 | 8 |
| Boar（野猪） | `B` | 120 | 中 | **领地反击**：被攻击 80% 反击，不去动它偶尔也冲过来 | 密林、丘陵 | 5 |
| Rabbit（兔子） | `r` | 20 | 极快 | **不可追击**：不出现在相邻格，只能陷阱/远程打（v1 不做远程→兔子不可狩猎，留生态意义） | 草原、疏林、沙地 | 12 |

### 1.2 生态种群

```
每个生态区维护一个 herd_map: HashMap<TerrainKind, Vec<Entity>>

每 50 tick（约 5 秒）→ 遍历所有生态区：
  对于 Deer 种群：
    当前数量 < max && 5% 概率 → 在该生态区边缘刷一只
  对于 Boar：
    当前数量 < max && 3% 概率 → 边缘刷新
  对于兔子：
    当前数量 < max && 8% 概率 → 边缘刷（兔子恢复最快）
  
  繁殖（v1 简化不追异性）：当前 < max && 1%/tick 在已有猎物附近 5 格内刷一只幼年体

幼年体标记：Cooldown 5000 tick（约 0.4 天）后变成年，成年前不屠宰
```

### 1.3 兔子作为生态引擎

v1 兔子不可狩猎（跑太快 + 没有远程）。兔子只做一件事——**养狼**。狼会时不时追兔子（不是追玩家）。兔子被狼杀了 → 尸体掉地上 → 可以捡（免费肉！）。捕食链自动运转。

## 二、尸体与屠宰

### 2.1 猎物死亡 → 尸体

```
猎物 HP ≤ 0 → despawn 猎物 Entity → spawn 尸体 Entity

Corpse 组件：
  animal: AnimalKind (Deer/Boar/Rabbit/Wolf)
  spoilage: u32 (= 90000 tick，约 7.5 天)
  butchered: bool
```

尸体 glyph：`%` 红色（DF 同款）。BlockMovement=true（挡路）。7.5 天不剥会自然腐烂消失。

狼尸体也走同样的路径——狼死后留尸体，也能剥。

### 2.2 屠宰操作

```
站尸体相邻格 → e → "屠宰" → 进度条 300 tick

工具需求：手持石刀/骨刀/木刀
屠宰速度：骨刀 1.8× > 石刀 1.0× > 木刀 0.6×
```

| 尸体 | 肉 | 骨 | 皮 | 脂肪 | 特殊 |
|------|:---:|:---:|:---:|:---:|------|
| 鹿 | ×3-5 | ×1-2 | ×1 | — | — |
| 野猪 | ×5-8 | ×2-3 | ×1 | ×1-2 | — |
| 兔子 | ×1-2 | — | ×1 | — | DF 同款——太小了 |
| 狼 | ×2-4 | ×1-2 | ×1(粗糙) | — | 毛皮粗糙→保暖值低（等衣物系统） |

### 2.3 屠宰产物

产物掉在尸体脚下 Pile 里。如果手有空位 → 自动捡。

### 2.4 新增物品

| ItemKind | 显示名 | glyph | 来源 | 用途 |
|------|------|:---:|------|------|
| RawMeat | 生肉 | `m` | 屠宰 | 可直接吃（腹泻风险）/ 烹饪 |
| CookedMeat | 熟肉 | `M` | 篝火烹饪 | 安全食用 |
| SmokedMeat | 烟熏肉 | `M` | 烟熏架 | 超长保质 |
| Leather | 皮 | `d` | 屠宰鹿/野猪 | 皮甲前置 |
| RoughLeather | 粗皮 | `d` | 屠宰狼 | 低保暖皮甲 |
| Fat | 脂肪 | `f` | 屠宰野猪 | 烹饪/灯油(以后) |
| RottenMeat | 腐肉 | `m` | 肉过期 | 吃了 50% 腹泻 |

## 三、腐败系统

### 3.1 腐败计时

所有带 `Perishable` 组件的物品实体每 tick 递减 `spoilage`。归零 → 变质。

```rust
struct Perishable {
    pub spoilage: u32,   // 剩余 tick
    pub total: u32,      // 初始总 tick（UI 显示百分比）
    pub variant: PerishKind, // RawMeat / CookedMeat / Corpse
}
```

| 种类 | 初始 spoilage | 实际时间 | 变什么 |
|------|:---:|------|------|
| 尸体 | 90,000 | ~7.5 天 | 腐肉 ×0-1（蝗虫/天气决定） |
| 生肉 | 180,000 | ~15 天 | 腐肉 → 吃了 50% 腹泻 |
| 熟肉 | 360,000 | ~30 天 | 腐肉 |
| 烟熏肉 | 1,200,000 | ~100 天 | 腐肉 |

### 3.2 烟熏架交互

```
手持生肉 × 1-5 → 站烟熏架相邻格 → e → "烟熏"
进度条 500 tick
需要篝火邻格（烟熏架必须在篝火旁 3 格内）
产物：烟熏肉 × N（N = 投入的生肉数量 × 0.8，20% 损耗）
```

烟熏架终于不空了。

## 五、AI 掠食（狼杀猎物）

### 5.1 狼的捕猎

现有狼 AI 的 `primary_target` 只追玩家/殖民者。新增：

```
50% 概率追玩家（跟现在一样）
30% 概率追最近的非狼猎物（鹿/野猪/兔子）
20% 随机晃荡
```

狼咬猎物 = 走现有 combat 管线 → 猎物死 → 留尸体 → 你可以捡漏。

### 5.2 猎物 AI

- **鹿**：见了任何 Hostile 实体 10 格内 → 朝反方向跑（跟狼的 Fleeing 一样）
- **野猪**：见了 Hostile 50% 无视 50% 逃跑，被攻击后 80% 反击
- **兔子**：见了任何实体 8 格内 → 跳开（随机传送到视野外半径 5 格的空地）
