## 零、设计摘要

在「砍树→石刀→削尖棍→火烤矛」的五步链之前，插入真正的旧石器打制流程。核心变化：

1. **打制系统**：小石头→石片、大石头+石锤→大石片+石片边角料
2. **大石片**：手掌大的薄片，是石刀/石斧/石锤/石铲的唯一主料
3. **木质工具**：便宜、低效、易损——空手也能干活，但慢到怀疑人生
4. **骨器**：骨刀锋利高速但脆弱，骨针是高级制品的前置
5. **荆棘藤**：新植物 Bramble，采摘藤条 25% 扎手
6. **生火**：石钻 + 引火物 → 不用篝火也能点火

## 二、打制系统

### 2.1 配方

| 产物 | 材料 | tick | 需要工具 | 最低光照 |
|------|------|:---:|------|:---:|
| 石片(SmallFlake) | SmallStone×2 | 200 | 无 | ≥1 |
| 大石片(LargeFlake) | BigStone×1 | 300 | 手持石锤 | ≥1 |
| 长木棍(LongStick) | Wood×1 | 100 | 手持石刀/木刀/骨刀 | ≥1 |
| 削尖长棍(WoodSpear) | LongStick×1 | 150 | 手持石刀/木刀/骨刀 | ≥1 |
| 绳子(Rope) | Vine×3 | 150 | 无 | ≥1 |

### 2.2 大石片打制产出

```
BigStone×1 + 石锤 → 大石片×1-2 + 石片×2-3（边角料副产品）
```

边角料自动掉到脚下 Pile。石锤不消耗。

### 2.3 石钻生火

```
手持石钻 + 引火物(Leaves×5) → R → 搓火
```

- 进度：400 tick
- 成功率：35%（失败浪费引火物）
- 成功 → 脚下生成篝火（Campfire 实体）
- 光照需求：≥1

## 七、空手采集

`harvest.rs` 的砍树/采矿逻辑：当没有手持工具时，伤害降到空手档：

```rust
let (dmg_lo, dmg_hi) = match kind {
    TargetKind::Tree => {
        if has_axe(actor) { (30.0, 60.0) }       // 石斧
        else if has_wood_axe(actor) { (8.0, 15.0) } // 木斧
        else { (1.0, 3.0) }                       // 空手
    }
    TargetKind::Boulder => {
        if has_hammer(actor) { (25.0, 45.0) }      // 石锤
        else { (1.0, 3.0) }                        // 空手
    }
    // ...
};
```

空手攻击接入 `combat.rs`：手持武器的 ItemKind 检查，没武器伤害 3-8，有武器查表。
