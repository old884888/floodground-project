# 血壤 · Bloodsoil — 游戏介绍

> "你建立在尸骨上的家园，终将成为你的坟墓。"

文字为核心的殖民地模拟游戏。你是一个流亡群体的"看不见的操控者"，在古文明遗弃的大陆上从零建家。表面是砍树种地打野人——**内核是对人性的全面压榨，以及系统对你干的事的无情报复。**

当前为终端原型阶段（Rust + hecs ECS + ratatui 终端 UI）。

---

## 一、地图与世界

### 500×500 瓦片世界
- 一次性生成到内存，包含以下地形：
  - 草地（默认）、水域（不可通行）、树木（可砍伐）、岩石（可开采）、灌木（可采摘）
- 玩家的营地位于地图中心 (250,250) 附近

### 日夜循环
- 一天 = 12,000 ticks（约20现实分钟，Fast=2分钟，Turbo=80秒）
- 四时段：夜晚(0-25%, 80-100%) → 黎明(25-30%) → 白天(30-75%) → 黄昏(75-80%)
- 开局时间：早上8点（33.3%），天光约10游戏小时
- 夜晚视野大幅受限（可见半径8格），白天50格
- 篝火照亮半径8-15格，无视昼夜
- 室内黑暗：房顶标记 + 窗口导光系统（白天向内5格）
- 墙面遮挡视线：Bresenham 射线检测，墙/门真正阻挡视线

### 统一光照五级系统
所有格子的可见亮度由 `tile_light(x,y)` 统一计算，叠加四种光源：

| 级别 | 名称 | 来源 | 效果 |
|:---:|------|------|------|
| 0 | 完全看不见 | 夜晚户外无源、室内深处 | 不可见，不可制作 |
| 1 | 昏暗 | 火把、篝火边缘、黄昏、窗口余光 | 制作速度 ×0.5 |
| 2 | 明亮 | 白天户外、篝火近旁 | 制作速度 ×1.0 |
| 3 | 太亮了 | 预留（核爆/超自然） | 制作速度 ×0.8 |
| 4 | 亮瞎眼 | 预留（原子弹/神降） | ×0 |

光源叠加公式：环境光(日夜) + 火源 LightSource{radius, brightness} + 手持火把(半径5) + 窗口导光

### 视口跟随
- 摄像机以玩家为中心，地图边缘自动 clamp
- 方向键/WASD/hjkl 移动

### 迷雾系统
- 三层状态：未探索 / 探索过但不在视野（暗色记忆） / 当前可见
- 已探索过的地形标记持久保留，离开视野后变暗但不消失

## 二、角色系统

### 玩家
- 初始属性：HP 100、饥饿 50、口渴 50、精力 50、心情 60
- 属性受 tick 衰减（按天速率缩放）
- 可装备双手物品（左手/右手各1，同种可堆叠）

### 殖民者
- 通过 F6 调试菜单生成
- 自主 AI：Idle 状态，支持多角色切换（`;` 或 `1`-`4` 键）
- 玩家死后自动接班控制殖民者

### 俘虏
- 通过 F6 调试菜单生成
- 特殊属性：will（意志力），可被刑讯（T 键）
- 刑讯会触发反噬反应系统（目击者心情下降 + 声誉下降）
- will 归零触发 CaptiveBroke 事件

### 狼（敌人）
- 行为树：
  - **逃跑**：独狼 HP≤60% 50%概率逃；有同伴 HP≤40% 必逃；逃出15格停止
  - **追击**：距离≤8格 → 80%追玩家 / 20%随机晃
  - **咬人**：相邻格（dist≤1）→ 伤害15~25，冷却1tick
  - **闲逛**：超出感知范围 → 纯随机移动
- 生成：全图4群，每群1-3只，离营≥30格，群间≥40格
- 玩家死后 primary_target 改追殖民者

### 物品种类
| 物品 | 符号 | 来源 | 用途 |
|------|:---:|------|------|
| Wood（木头） | `=` | 砍树（C键） | 建材/燃料 |
| BigStone（大石头） | `O` | 挖矿（M键） | 建材/工具材料 |
| Stick（木棍） | `/` | 地面采集/灌木 | 工具材料 |
| SmallStone（小石头） | `o` | 地面采集/灌木 | 工具材料 |
| Berry（莓果） | `*` | 采摘结果期灌木 | 食物（E键吃） |
| 石刀 | — | 制作（R键） | 工具/材料 |
| 削尖棍 | — | 制作（R键） | 工具/材料 |
| 矛（火烤矛） | — | 制作（R键） | 武器 |
| 石斧 | — | 制作（R键） | 工具 |
| 火把 | — | 制作（R键） | 光源（半径5） |

### Pile 物品系统
- 每格地面最多128种不同物品（同种堆叠）
- e → 方向 → 弹窗浏览 → ↑↓选择 → G捡取
- D键丢弃手中物品（右手优先）
- 砍树/挖矿/采摘掉落的物品自动合并到脚下 Pile
- Pile 空壳自动清理

### 动作锁定
选择方向后自动重复执行砍/挖/采/刑讯/砸墙直到：
- 目标消失
- 按其他方向键移动
- 按 Esc 取消

## 四、制作系统

### 配方菜单
R 键打开制作弹窗 → ↑↓选择配方 → Enter 确认 → 自动收集材料 → 进度条制作

### 材料搜索
曼哈顿 dist ≤ 2（13格菱形），优先级：双手 → 脚下 → dist=1 → dist=2

### 旧石器配方
| 配方 | 材料 | 产物 | 进度 | 需篝火 | 最低光照 |
|------|------|------|:---:|:---:|:---:|
| 石刀 | 小石头×2 | 石刀 | 300 | 否 | ≥1 |
| 削尖棍 | 石刀+木棍 | 削尖棍 | 200 | 否 | ≥1 |
| 火烤矛 | 削尖棍 | 矛 | 400 | 是 | ≥0 |
| 石斧 | 大石头+石刀+木棍 | 石斧 | 800 | 否 | ≥1 |
| 火把 | 木棍 | 火把 | 20 | 是 | ≥0 |

### 制作动画
- 制作中弹窗显示旋转动画（⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏）+ 进度条 + 百分比
- 光照不足暂停制作但不取消（显示"太暗了…"）
- 篝火邻格自动检测

### 半成品系统（CraftWip）
- Esc 中断制作 → 生成 CraftWip 实体（黄色 `…` glyph）→ 保留进度
- 站到半成品格上按 R → 显示 [续作] → 从上次进度继续，不消耗材料
- progress=0 不生成 WIP；部分配方不可中断（can_interrupt=false）
- X 观察可见半成品名称和进度

## 五、村庄与建筑

### 村庄生成（F6 → 生成聚落）
- CDDA palette ASCII 模板生成
- 3种规模：小村（3-5房环篝火）、中村（5-8沿街）、大村（8-12围栏+大门）
- 5种建筑模板：

| 建筑 | 尺寸 | 特征 |
|------|------|------|
| 小屋 | 3×4~4×4 | 床+桌子，1扇窗 |
| 中屋 | 4×5~5×5 | 床+桌子+壁炉，2-3扇窗 |
| 大屋 | 5×6~6×7 | 床×2+桌子+壁炉+储物箱，3-4扇窗 |
| 工坊 | 4×4 | 工作台+熔炉+储物箱 |
| 仓库 | 3×3 | 储物箱×2-3 |

- 生成特点：随机旋转（4朝向）、材质替换（15%概率石墙替换木墙）

### 建筑组件
| 组件 | 特性 |
|------|------|
| 木墙/石墙 | 阻挡移动+视线，可砸毁掉落材料 |
| 门 | 可开关（开关切换阻挡状态） |
| 玻璃窗 | 透光+窗口导光 |
| 木床 | 可睡觉回复精力 |
| 储物箱 | 可打开浏览+存取物品 |
| 木地板 | 装饰性地面 |

### 室内黑暗系统
- 房顶标记（has_roof）的格子默认不可见
- 可见条件：窗导光（白天向内3格递减）+ 内部光源
- 墙体阻挡视线（BlocksVision），室内外视线隔离

### 玩家交互
| 操作 | 方式 | 效果 |
|------|------|------|
| 开门/关门 | e → 方向 → 选择 | 切换门阻挡状态 |
| 睡觉 | e → 方向 → 选择 | 精力 +50 |
| 翻容器 | e → 方向 → 选择 | 显示容器内物品列表 |
| 砸墙 | e → 方向 → 选择 → 锁定方向 | 掉木头/石头，墙消失 |

### 玩家建造（B 键）

CDDA 式逐格建造：B 键 → 选配方 → 选方向（邻格）→ 进度条 → 实体生成。全程复用制作系统的材料搜索和进度 UI。

| 建造项 | 材料 | 耗时 | 产物 |
|------|------|:---:|------|
| 木墙 | Wood×3 | 4s | WoodWall HP500，阻挡移动+视线 |
| 石墙 | BigStone×3 | 8s | StoneWall HP1200 |
| 木门 | Wood×2 | 3s | Door，e 键开关 |
| 木屋顶 | Wood×3+Stick×2 | 2s | 脚下 set_roof，需邻墙支撑 |
| 尖刺陷阱 | Stick×3 | 1.5s | StickTrap，仅建造者可见，踩上 20-40 伤害后消失 |

- 建造中自动加速到 Fast，完成/取消恢复原速
- Esc 取消建造 → 材料不退
- 屋顶支撑检测：四邻至少有一个墙或已有屋顶

### 侧边栏 Tab

### 观察系统（X键）
- X键 → 选方向 → 固定观察目标
- 侧栏显示诙谐描述文本（覆盖所有地图元素）
- Pile 全展开：每种子项"名称 ×数量" + 缩进俏皮话
- 生物用文字伤势标签（轻伤的/负伤的/重伤的/濒死的）替代血条数字
- 再按 X 或移动自动清除

### 弹窗叠加
- **物品弹窗（e键）**：列出地面所有物品，↑↓选择 G捡取

### 速度控制
| 操作 | 效果 |
|------|------|
| 空格 | 步进一 tick / 暂停 |
| +/= | 加速：暂停 → 步进 → 正常 → 快进 → Turbo(15x) |

## 八、刑讯与反噬系统

### 刑讯（T键）
- 走到俘虏相邻格 → 按 T → 刑讯一次
- 俘虏 will -= 掷骰(15~25)
- will ≤ 0 → CaptiveBroke 事件（日志 + 声誉再跌）

### 反噬反应
- 目击者判定：所有 Colonist 与刑讯坐标曼哈顿距离 ≤ 6 且视线未被阻挡 → 算目击
- 目击者 Mood -= 掷骰(8~15)，写入叙事日志
- 声誉 Reputation 每次刑讯 -5，俘虏崩溃再 -10
- 声誉低于阈值 → 日志出现警告色文本
- 目击者心情跌破 30 → 情绪不稳警示

## 十二、叙事系统

三层文本管线（当前实现 L1）：

- **L1 即时日志**：事件 → 中文模板文本（"你挥斧砍向那棵老树。"）
- **L2 辐射影响**（未实现）：事件影响扩散到周边角色
- **L3 记忆沉淀**（未实现）：重要事件写入角色记忆，长期影响行为

文本风格：诙谐俏皮 + 冷不丁的沉郁——"这游戏表面跟你开玩笑，内核在解剖你。"
