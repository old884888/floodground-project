# 设计约束

> 别问我为什么这么定。就是这么定的。阿弥陀佛。

---

## 一、空间与视野

| 约束 | 值 | 备注 |
|------|-----|------|
| 可见距离公式 | 曼哈顿 `|dx| + |dy|` | CDDA 用欧几里得，我们用曼哈顿。菱形就菱形，Brogue 祖传。 |
| 白天可见半径 | 999（全图） | 500×500 地图，999 约等于全图可见 |
| 夜晚可见半径 | 8 | 时刻表：黎明 0-10% / 白天 10-60% / 黄昏 60-70% / 夜晚 70-100% |
| 篝火照亮半径 | 15（曼哈顿） | 营地周围始终温暖，火光照亮无视昼夜 |

## 二、物品系统

| 约束 | 值 | 备注 |
|------|-----|------|
| 双手容量 | 每只手 1 个物品 | 无背包，两只手就是全部库存 |
| 地面一格 stack 上限 | 128 种不同物品 | 同种可堆叠（字段 `count: u32`） |
| 地面物品捡取方式 | `e` → 方向选格 → popup 列物品 → `↑↓` 选 → `G` 捡 | 不可直接 `G` 盲捡地面 |
| 物品丢弃方式 | `D` 丢脚下/邻格 | 如果脚下有空格优先脚下，否则找邻格空地 |
| 物品种类（当前） | Wood / BigStone / Stick / SmallStone / Berry | Planmini1 范围 |

## 三、UI 布局

| 约束 | 值 | 备注 |
|------|-----|------|
| 布局 | 顶栏 / 地图+侧栏(68:32) / 日志 / 帮助栏 | ratatui 三区结构 |
| 侧边栏 | Tab 翻页，`[` / `]` 切换 | 最多预留 6 个 tab |
| 侧边栏-界面一 | 角色属性（HP/饥饿/精力/心情/意志/性格） | 默认页 |
| 侧边栏-界面二 | 双手物品 + 估算价值 | 未来扩展 |
| 侧边栏-界面三 | 营地状态 | 未来扩展 |

## 四、交互键位

| 按键 | 动作 | 备注 |
|------|------|------|
| 方向键 / WAS(L右移) | 移动 | D 被丢弃占了，右移用 L 或方向键 |
| `G` | 捡取 / 采摘 | 普通模式下直接捡脚下；inspect 模式下列表中捡 |
| `D` | 丢弃手中物品 | 右手优先 |
| `C` | 砍伐邻格树 | 每次 30-60 伤害 |
| `M` | 开采邻格岩石 | 每次 20-45 伤害 |
| `T` | 刑讯邻格俘虏 | 触发反噬系统 |
| `e` | 进入/退出地面 inspect 模式 | 再按方向选格，按 `.` 看脚下 |
| `Tab` | 切换选中角色 | 面板显示对应角色信息 |
| `[` / `]` | 侧边栏翻页 | 左 / 右 |
| 空格 | 步进一 tick | 暂停模式下用 |
| `+` / `-` | 调速 | 暂停/步进/正常/快进 |
| `Q` | 退出 | |
| `[` / `]` | 调试日夜快进 | Planmini1 调试用 |

## 五、世界物理

| 约束 | 值 | 备注 |
|------|-----|------|
| 地图大小 | 500 × 500 瓦片 | 一次性生成到内存 |
| 一天长度 | 12,000 ticks (100ms/tick) | 约 20 现实分钟一天 |
| 需求衰减 | 按天速率缩放 | 饥饿 ~30/天，避免和 MVP 一样几分钟饿死 |
| 实体阻挡 | BlockMovement + BlockVision marker | 树、岩石、篝火有 BlockMovement |
| 营区大小 | 12×12 (camp_half=6) | 中心 (250,250) 附近 |

---

## 六、黑暗系统（MVP 遗留 + Planmini1 保留）

| 约束 | 值 | 备注 |
|------|-----|------|
| 刑讯 | 按键 `T`，俘虏 will -= 15~25 | 唯一黑暗操作（当前） |
| 反噬 | 目击者（≤6格）情绪下降 + 声誉下降 | 系统记账 |
| 声誉 | i32 全局值，跌破 -10 变红警告 | 视觉反噬先行 |

---

## 七、技术约束

| 约束 | 值 | 备注 |
|------|-----|------|
| ECS 框架 | hecs | 单 crate，编译秒过 |
| UI | ratatui + crossterm | 纯终端 |
| 数据 | serde + ron | terrain.ron / actors.ron |
| 日志 | Vec<String>，上限 200 行 | 老日志自动滚出 |
| 视口跟随 | 摄像机以玩家为中心，边缘 clamp | 地图不满屏时钉在 (0,0) |
| 鼠标 | 全吞不处理 | 以后留给格子选择 |
