## 一、✋ 黑暗操作完全体

当前只有刑讯。刑讯是幼儿园。

### 1.1 🫁 器官收割

俘虏不是资源——**俘虏身上的器官是资源**。

```
俘虏 → 手术台（石刀+石钻+骨针）→ 指定器官 → 进度条 → 
  产物：肾/心/肝/肺/眼球/皮肤
  效果：俘虏 −HP（大量），可能死亡，活下来的忠诚清零
  反噬：目击者心情暴跌，同族殖民者可能叛变
```

- **器官移植**：殖民者缺肾？俘虏身上摘一个换上。HP 上限 +20，感染概率 15%
- **器官交易**：等势力系统到位——器官是硬通货，矮人出价最高，精灵拒绝交易
- **器官品质**：年轻人 > 成年人 > 老年人，俘虏意志越高品质越好

### 1.2 🦴 血锻祭品（矮人专属）

```
活祭 → 熔炉 → 传奇武器
品质 = 祭品意志 × 种族纯度 × 仪式时间
```

矮人殖民者把俘虏推进熔炉——产出带名字的传奇装备。伤害 +50%，附带诅咒词条（"持有者每晚 HP −1"）。这是矮人唯一的终极武器来源。

### 1.3 🌳 根化（精灵专属）

```
俘虏 + 精灵仪式 + 肥料（骨头/尸体）→ 活体建筑
```

精灵把俘虏种进土里——人变成会呼吸的树墙。HP 自动回复、受伤会流血（树汁）、冬天休眠。拆毁时掉落骨头和皮——而不是木头。

### 1.4 👻 灵魂提取（龙裔专属）

```
俘虏 → 灵魂提取台 → 灵魂晶石 + 空壳奴隶
```

龙裔抽出灵魂——晶石是终极魔法资源（照明/传送/附魔），肉体变空壳——不吃饭、不睡觉、不违抗命令、不会心情波动。唯一的代价：晶石碎了空壳就死了。

### 1.5 🧟 肉傀儡（哥布林专属）

```
尸体 + 哥布林仪式 → 肉傀儡
```

哥布林把死掉的同伴/俘虏做成战斗傀儡——移动慢、伤害高、免疫心情。不打仗时能搬东西。唯一的问题是会腐烂。

### 1.6 💀 灵魂解离（不死者专属）

```
灵魂晶石 → 亡灵精华 → 禁忌科技研究 → 亡灵科技
```

不死者不搞花里胡哨的。他们把灵魂晶石拆解成精华，用于亡灵科技研究。每拆一个晶石 = 100 研究点。效率最高的科技路径——也是最贵的（一个晶石 = 一个活人）。

## 二、🏰 营区帝国

### 2.1 🏗️ 从营地到王国

```
冒险模式 → 划地建营 → 营地升级 → 城镇 → 王国
```

营地不是独立场景——是世界坐标上的实地区域。敌人能走到门口撞墙。

| 阶段 | 人数 | 建筑 | 解锁 |
|------|:---:|------|------|
| 营火 | 1-5 | 窝棚/地坑 | 基础生存 |
| 营地 | 5-15 | 木屋/工坊/围墙 | 种植/畜牧 |
| 村庄 | 15-40 | 石屋/瞭望塔/铁匠铺 | 铁器/势力外交 |
| 城镇 | 40-100 | 宫殿/市场/学院 | 科技/法律/赋税 |
| 王国 | 100+ | 城堡/军队/监狱 | 对外战争/附庸 |

### 2.2 🏛️ 营区建设

```
园区规划：
├── 生产区：农田/牧场/伐木场/矿场
├── 居住区：住房/食堂/医疗所
├── 军事区：兵营/训练场/武器库
├── 牢房区：刑讯室/手术室/囚室
├── 祭祀区：种族仪式台/祭坛/墓园
└── 交易区：商队站/仓库/货币兑换
```

每一区有专属建筑和配方链。牢房区的囚室关俘虏——每天自动喂食（或不喂）、自动刑讯（如果设了值班）。祭祀区的精灵根化仪式、矮人血锻熔炉、龙裔提取台——每个种族有专属建筑和升级树。

### 2.3 ⚖️ 法律系统

```
颁布法律 → 触发条件 → 自动执行 → 影响心情/忠诚/生产
```

| 法律 | 效果 | 反噬 |
|------|------|------|
| 食人合法 | 尸体→肉，饥饿解决 | 非哥布林殖民者心情 −20 |
| 奴隶制 | 俘虏强制劳动 | 每 10 天有人逃跑/反抗 |
| 宵禁 | 夜晚不外出，安全 +50% | 心情 −5 |
| 配给制 | 食物消耗 −30% | 饥饿时心情 −15 |
| 活祭 | 每季献祭一人，神恩 +100 | 目击者 −30，可能叛变 |
| 自由恋爱 | 殖民者心情 +5 | — |
| 强制婚配 | 人口增长 +50% | 被配的 −20 |

## 三、⚔️ 战争与征服

### 3.1 🗺️ 势力战争

```
世界观六大种族 × 四种意识形态 → 每局 6-12 个势力，各有定居点和初始关系

人类扩张帝国 → 矮人熔炉氏族 → 精灵古树联邦 → 龙裔龙脊王朝 → 哥布林碎骨部落 → 不死者灰烬议会
```

- **宣战**：外交恶化 → 下战书 → 部队出发
- **攻城**：带攻城器械（冲车/云梯/投石机）砸城门
- **守城**：围墙+箭塔+陷阱+城门。敌人能烧你的田
- **附庸**：打赢不灭国 → 签条约 → 每季进贡 → 成为你的卫星国
- **灭国**：占领所有定居点 → 势力消失 → 种族成员沦为无国籍流民

### 3.2 🪖 军事编制

```
部队 = 角色 + 装备 + 训练等级 + 士气
```

- **训练**：兵营里每天操练 → 武器技能 +1% → 满级 100% 伤害加成
- **装备**：矛兵/剑兵/弓兵/弩兵/骑兵。装备品质决定伤害和防御
- **士气**：赢了涨、输了跌、饿了崩、死了同伴大崩
- **军官**：指派殖民者当队长 → 队长属性加成全队

## 四、💑 伦理剧 & 后宫

### 4.1 💕 社交关系网

```
每个角色维护一张关系图：
  { entity_id: { trust, affection, resentment, rivalry, romance } }
```

- **信任**：一起干活、一起打架 → 涨。撒谎、背刺 → 跌
- **好感**：送礼物、救过命、一起睡过觉 → 涨。摘了他的朋友器官 → 暴跌
- **怨恨**：你欺负过他、杀过他朋友、刑讯过他 → 积累到阈值 → 背刺概率 UP
- **情敌**：两个人同时喜欢一个人 → 互相扣好感 → 可能在夜里打架

### 4.2 💍 后宫线

```
殖民者A 好感度 > 80 + 赠送戒指（骨制/石制/金属）→ 求婚 → 成为配偶

配偶 buff：
  +10 心情（持续）
  +20% 工作效率（在彼此视野内）
  一方死了 → 另一方 −50 心情持续 10 天（守寡期）

多配偶制：
  可以娶/嫁多个（看法律）
  每多一人 → 每人心情微降（嫉妒叠层）
  超过 3 人 → "后宫不宁"随机事件：打架/下毒/出走
```

### 4.3 🎭 伦理事件

| 事件 | 触发条件 | 结果 |
|------|------|------|
| 出轨 | A 好感低 + B 好感高 + A 和 B 互不知情 | 发现后信任清空 + 怨恨 + 配偶可能自己动手 |
| 三角恋 | 两人同时喜欢第三人 | 好感互扣 → 可能决斗 |
| 私生子 | 两个独处 + 好感 > 60 | 娃出生，如果已婚非配偶 → 丑闻 |
| 复仇 | 怨恨 > 80 + 武器在手 + 夜班独处 | 直接在睡眠中攻击目标 |
| 殉情 | 配偶死 + 心情 < 10 | 角色自杀（概率 15%） |
| 背刺篡位 | 殖民者怨恨玩家 + 忠诚 < 20 + 你受伤 | 在你睡觉时攻击你 |
| 流言 | 随机触发 | 随机扣两个角色之间的信任 |

### 4.4 👶 生育 & 世代

```
配偶 + 同床 + 好感 > 50 → 怀孕概率 / tick

孕期：30 游戏天
婴儿期：10 天（不能干活，需要喂养）
儿童期：30 天（轻量劳动，学习技能）
成年：40 天后 → 新殖民者，继承父母两项最高技能 × 0.6
```

## 五、🧬 种族深度

### 5.1 种族专属科技树

每个种族有**自己的科技树**，不是换皮。人类走"效率+外交"路线，矮人走"锻造+战争"——每条树 15-20 个节点，从旧石器到铁器到蒸汽。

### 5.2 种族歧视 & 融合

```
多种族营地：
  + 技能多样性（精灵种田 + 矮人锻造 = 效率 MAX）
  − 社交摩擦（精灵看不起矮人，矮人受不了精灵的高傲）
  − 食物冲突（精灵只吃素，哥布林吃腐烂的，人类吃正常的）
```

### 5.3 种族转换

```
俘虏 → 刑讯 → will 归零 → "转化"
转化 = 变成你的殖民者，但保留原种族特性
```

你囚禁了一个精灵？意志打碎之后，他就是**你的精灵殖民者**——带着精灵的农业技能加成，但和你的矮人工匠处不来。

## 六、🔮 超自然 & 遗迹

### 6.1 造物者遗迹探索

```
地表废墟 → 地下 1-3 层 → 地下 4-6 层 → 核心层
  基础物资     科技蓝图      传奇装备    造物者语言
  文献碎片     灵魂晶石      禁忌知识    终极科技
```

每层有独特怪物和陷阱。守卫机械（地表）、变异生物（中层）、不可名状之物（核心）。

### 6.2 禁忌科技

```
亡灵精华 → 研究台 → 禁忌科技节点

复活术：尸体 → 活人（带记忆衰减）
永生：殖民者停止衰老（代价：不能再生育）
传送：两个营地之间瞬间移动物品
诅咒：远程杀死指定敌人（代价：自己 HP −50%）
```

每项禁忌科技有伦理惩罚（声誉暴跌 + 全体殖民者心情下降）——**除非你已经让所有人都麻木了**。如果营地里已经发生了足够多黑暗操作，殖民者逐渐对禁忌科技脱敏。

## 七、🎯 终极愿景

你是一个流亡者群体的看不见的操控者。

开始你在营火旁，手无寸铁，饿着肚子摘莓果。

你活下来了。
你学会了打制石器。你建了窝棚。
你抓了俘虏。你第一次刑讯了他。

然后你发现——**这比种地高效**。

俘虏不只是劳动力。他们是器官库、是祭品、是灵魂原料、是活体建筑材料。
你开始批量抓人。你建了牢房区。你设了值班表自动刑讯。

然后你发现——**邻居也不老实**。

矮人氏族来抢你的矿。精灵联邦谴责你的"人道灾难"。哥布林部落趁夜偷你的粮。

你砍回去了。你打赢了。你附庸了他们。

然后你发现——**附庸还不够**。

你开始灭国。一国一国地消失。你的国王称号换成了皇帝。你的王国版图从 500×500 扩到整个生态区——六种生态区、六种征服。人类虚伪、矮人直率、精灵高傲、龙裔孤僻、哥布林欺软怕硬、不死者不可理喻——全跪在你的脚下了。

然后你抬起头。

造物者遗迹核心层的入口在你脚下。

你走进去。

门关上了。
